# Proyecto SQL - Tienda en línea

## Descripción
Proyecto de análisis de una base de datos simulada de una tienda en línea usando SQL.

## Objetivo
Analizar clientes, productos y ventas para obtener información útil para la toma de decisiones.

## Contenido del proyecto
- Creación de tablas: clientes, productos, ventas  
- Inserción de datos  
- Consultas SQL  
- Uso de JOIN para relacionar tablas  

## Análisis realizados
- Clientes con mayor volumen de compras  
- Productos más vendidos  
- Ventas por mes  

## Herramientas
SQL (SQLite / MySQL)
